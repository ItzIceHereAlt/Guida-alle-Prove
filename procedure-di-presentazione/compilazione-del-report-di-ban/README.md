# Compilazione del Report di Ban

La fase finale di un controllo con esito positivo è la compilazione e la pubblicazione del report di ban. Questo documento è l'atto ufficiale che formalizza la sanzione e presenta le prove alla leadership del team per la revisione.

Un report incompleto, poco chiaro o non conforme agli standard può indebolire un ban altrimenti valido e creare problemi in fase di appello. La professionalità si dimostra non solo nel trovare le prove, ma anche nel documentarle correttamente.

Questa sezione fornisce le linee guida e un template di riferimento per la compilazione di un report di ban professionale e inattaccabile.
